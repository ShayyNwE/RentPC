# RentPC-Home Website
Welcome to the RentPC-Home website! This repository contains the HTML and associated files for the RentPC-Home web application.

## Table of Contents
- [Description](#description)
- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)

## Description
RentPC-Home is a website for renting gaming PCs, monitors, and setups. Users can explore different products, add them to their cart, and contact the company for inquiries.

Features
* Navigation through different sections: Setups, PCs, Monitors, Outputs/Inputs, and Contact Us.
* User authentication and registration.
* Shopping cart functionality.
* Search bar for product search.
* Information about the company's services and contact details.
* Social media links for following RentPC on Instagram, Facebook, and LinkedIn.

## Installation
1. Clone the repository: `git clone [https://github.com/ShayyNwE/RentPC]`
2. Open the project in your preferred code editor.

## Usage
1. Open the `index.html` file in a web browser to explore the website.
2. Navigate through different sections using the navigation bar.
3. Register/login to access additional features such as the shopping cart.
4. Explore products and add them to the cart.
5. Contact RentPC using the provided contact information.
6. Follow RentPC on social media for updates.
